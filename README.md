## Description

# NestJS Google OAuth Authentication

A NestJS project that enables users to sign up or log in using their **Google account** via OAuth2. It includes:

- Google OAuth2 Login with Passport
- JWT Authentication
- MongoDB Integration (via Mongoose)
- REST API endpoints
- Token-based redirection

Running the App

# Development
npm run start:dev

# Production
npm run build
npm run start:prod

# API Endpoints
 
Auth Routes

Method	Route	Description
POST	/auth/signup	User signup with email/password
GET	/auth/google/login	Start Google OAuth login
GET	/auth/google/callback	Google OAuth callback
